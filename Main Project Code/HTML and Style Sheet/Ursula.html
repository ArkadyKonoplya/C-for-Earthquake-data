Welcome to Project Ursula

Ursula the Earthquake Slayer, Ursula is a fictional character from Walt Disney, The little Mermaid. Ursula was born with great powers, just like earthquakes.Project Ursula is a C# and SQL application used to manipulate data related to earthquakes.

Earthquakes The earth consists of many major and minor tectonic plates. These plates around the earth are constantly moving, but when there is movement along faults and plate boundaries, that is when earthquakes will happen. The specific data that will be presented in this project is from the country of Canada and includes categories including the dates ranging from the year 1985 to 2019, latitude, longitude, depth, magnitude, magnitude type and the place (province(s)) which were affected. The data that will be viewed and manipulated to fit “Team Ursula’s” Capstone project for the winter 2021 COMP 6036 term.

The toolset's primary purpose is the tracing of maps and processing of Polygons and Polylines.

  • A toolset to trace Google Map Features
  • Overlay Vector Conversion and Storage
  • CRUD Complete for Markers, Polygons and Overlays/Heatmaps
